A **backend .NET MVC project** for managing company operations, designed to support multiple roles with secure access. This project demonstrates full-stack development with a focus on **enterprise-level CRUD operations** and **role-based access control**.  

**Project Highlights:**
- Developed using **C#, .NET MVC, and EF Core** to provide a robust backend structure.  
- Integrated a **SQL Server database** to store employees, departments, and role information.  
- Implemented **CRUD operations** (Create, Read, Update, Delete) for both employees and departments.  
- Developed **role management system** with Admin and User roles, including permission-based access to pages and operations.  
- Built a **security module** to ensure that sensitive data is protected and only accessible by authorized users.  
- Focused on **scalable and maintainable code architecture**, allowing easy addition of new features or user roles.  
- Delivered a **live demo** to showcase real-time functionality and system workflow.  

**Key Features:**  
- Employee and Department management (CRUD)  
- Admin/User roles with permission control  
- Secure login and authentication system  
- SQL Server database integration
